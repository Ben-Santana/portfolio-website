## Overview

This project is an inexpensive lighting and video control platform that transforms any space into an interactive audiovisual experience for electronic music. Traditionally, these sorts of systems cost tens of thousands of dollars—requiring expensive lights and a proprietary DMX controller like a GrandMA console. Using a Novation Launchpad along with custom hardware and software, this system allows a user to instantly trigger effects on addressable LED strips, video projectors, and RGB laser scanners.

This allows for an immersive performance space for electronic music to easily be set up in any location, where the DJ and LD to perform together in real time.

The system is powered by six addressable LED ropes, an ultra short throw laser projector, two custom RGB laser scanners, and many ESPs controlling all the hardware. There is a dedicated WiFi network for the ESPs to communicate to a laptop running a custom application for a Launchpad connected over USB.

## Features

- **Launchpad control** - Intuitive grid-based interface for triggering scenes and effects
- **Synchronized Lighting** - Addressable LED strips respond in real-time
- **Video Integration** - Coordinated video playback for full room immersion
- **Scene Presets** - Pre-programmed lighting and video combinations
- **Live Control** - Real-time manipulation of colors, patterns, and effects
